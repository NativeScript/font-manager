# @nativescript/font-manager

```javascript
npm install @nativescript/font-manager
```

A `FontFace` / `FontFaceSet` (Web Font Loading API) implementation backed by native code on each
platform: Objective‑C (iOS), Kotlin (Android) and C++/WinRT (Windows).
